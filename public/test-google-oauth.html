Google OAuth Test

Current Configuration:

Backend URL: https://api.fadsms.com/api/auth/google

Callback URL: https://api.fadsms.com/api/auth/google/callback

Frontend URL: https://fadsms.com/auth/callback

Current Issue:

Error: "Missing required parameter: code"

Cause: The redirect URI in Google Cloud Console doesn't match the backend callback URL.

Test Google OAuth:

Test Google OAuth Flow

What to do:

  1. Go to Google Cloud Console
  2. Navigate to APIs & Services > Credentials
  3. Find your OAuth 2.0 Client ID
  4. Edit the client
  5. In "Authorized redirect URIs", make sure you have exactly:
  6. https://api.fadsms.com/api/auth/google/callback
  7. Save the changes
  8. Wait a few minutes for changes to propagate
  9. Try the Google OAuth flow again

Expected Flow:

  1. Click "Test Google OAuth Flow" → Redirected to Google
  2. Authenticate with Google → Google redirects back with authorization code
  3. Backend processes the code → Creates/finds user → Generates token
  4. Backend redirects to frontend with token → User is logged in
